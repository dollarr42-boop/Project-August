PROJECT AUGUST

iPhone setup:
1. Upload all files in this folder to a static web host such as GitHub Pages, Netlify, or Cloudflare Pages.
2. Open the hosted URL in Safari.
3. Tap Share > Add to Home Screen.

The app stores data locally on the device. Use History > Export backup regularly.

Included:
- Countdown to August 13, 2026
- Six lifting days plus Sunday recovery
- Cardio, protein, water, steps, sleep, creatine, and alcohol tracking
- Exercise set logging with weight, reps, and RPE
- Nutrition and body measurement logs
- Daily compliance score and streak
- JSON backup export
